## What is it?
Commander is shortcut for Apple and you can modify it (add more commands)
## Features (commands)
- !help *Shows all the commands*
- msg me *Sends message to you*
- msg *Sends message to someone*
- app *Open any app*
- There is more commands!
## Modification
